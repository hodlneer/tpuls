Colors

Use variables or classes for colors.

Background colors have a -bg suffix.

To incorporate a new color, add the color variable to the variables partial and refer to it in the Sass map inside the colors partial.

Reference files: _colors.scss, _variables.scss

Typography

Typography consists of helper classes for text styling and basic styling for HTML elements like headings, lists and paragraphs.

Reference files: _functions.scss, _typography.scss, _variables.scss

Typography: Font Families

Use a variable or class for font families.

Roboto Slab is the default font for the project and is available via Google Fonts.

$font-family-serif
